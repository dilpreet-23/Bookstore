# 📚 Bookstore - Beyond Pages
### 
A full-stack online bookstore application built to provide users with a seamless book shopping experience. This app supports user registration, browsing books, and managing content using modern web technologies.
***
## 🚀Features
* 🔐 User Authentication (Login & Register)
* 📖 Browse & View Book Details
* 🛒 Add and Manage Books (Admin)
* 💡 Responsive UI with React
* 🌐 RESTful API with Node.js & Express
* 🧠 MongoDB Database Integration
***
## 🧰 Tech Stack
#### Frontend
* React.js
* Axios
* React Router DOM
* CSS Modules / Tailwind (if used)
#### Backend
* Node.js
* Express.js
* MongoDB with Mongoose
* JWT (JSON Web Token) for Auth
* Bcrypt for Password Hashing
